# Termux-Lock
![Termux-Lock preview](Termux-Lock.jpg)

## introduction
Create a lock in Termux that only you have access to.

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ apt update -y && apt upgrade -y
$ pkg install python -y
$ pkg install git
$ git clone https://github.com/Whomrx666/Termux-Lock.git
$ cd Termux-Lock
$ chmod +x setup.py
$ chmod +x Termux-Lock.py
$ python setup.py
$ python Termux-Lock.py 
```

## Instructions
- **first**: install tools according to the instructions above
- **second**: Register first in the register menu
- **third**: Continue by logging in, enter the username and password you previously created
- **Last**: Congratulations, you have successfully locked your Termux
